# Internship Task 2 - Portfolio Website

## 🔥 Project Overview

This project is a responsive portfolio website built using:

* Nunjucks (Templating Engine)
* Vite (Build Tool)

## 📁 Folder Structure

* src/templates → Nunjucks templates
* dist → Final compiled output
* index.html, services.html, contact.html → Entry files

## 🚀 How to Run

```bash
npm install
npm run dev
```

## 🏗 Build Project

```bash
npm run build
```
